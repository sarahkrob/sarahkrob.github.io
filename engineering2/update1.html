Engine System Update 1


So far for my engine system I have made a new project and set up basic moment and rotation. This is essentially recreating the existing rigidbody but in a format that’s compatible with the functionality I want to add later. I also moved the user input information for physics objects into my library, so that I can potentially allow the user to input the keys they would like to use to control the object without having to go and swap all of the keys themselves. 


Currently, my movement and rotation are just straight velocity affected by time. My next steps are to add drag to the linear movement in order to create acceleration, and torque to the rotation in order to create angular acceleration. Eventually I would like to be consolidating forces into vectors and creating moments of inertia so that forces can be applied at specific locations on the object-- this will require me to know the dimensions of the object, so it’s my next major goal after getting basic acceleration working. 


My biggest hangup so far has just been trying to figure out how to set everything up so I’m able to add more complex functionality later. I think I have at least a basic understanding of what I need to do for now, so I should just be able to program the actual physics without issue from here on out (famous last words).
