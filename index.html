Paradox

C# & Unity -- Spring 2019 -- Nintendo 64-style action game created for class. Programmed by me with art assets from other team members. Paradox is a top-down shooter action game in which the player must clear waves of enemies in order to collect items that will allow them to travel through time. Occasionally, one of the player's "past selves" will spawn, which they must avoid damaging. The game features three levels, a full soundtrack and effects, and basic enemy AI. More Images

Coin Collector

C++ & Godot -- Fall 2019 -- Currently in-progress 3D platformer for a 3-person team project in CS354R Game Tech. Features a variety of movement systems-- walking, jumping, dashing, various ledge interactions and interactions with slopes, and picking up objects. It can be played alone or over a network with up to 4 players at a time competing to pick up coins as fast as possible. More Images

Non-Photorealistic Shader Demonstration

C++ & OpenGL -- Fall 2018 -- Final project for CS352 Graphics. Showcases a variety of non-photorealistic rendering techniques such as cel shading and Gooch shading with a fully interactive interface and 3D camera movement. More Info

Ray Tracer

C++ & OpenGL -- Fall 2018 -- Ray Tracer project for CS352 Graphics. A fully-functioning ray tracer with Whitted Illumination Model, Phong shading, reflection & refraction, triangle mesh intersection, Phong interpolation, cube mapping and antialiasing. For efficiency, it utilizes kd trees and multithreading. This project was built on basic starter code, but all main functions were implemented by me. More Images

Python & SDL -- Recreated Space Invaders from scratch in Python using SDL, a library that provides access to game functions such as rendering, audio and keyboard input. Includes all normal Space Invaders features as well as networking functionality with theoretically unlimited players able to join at once. More Info
